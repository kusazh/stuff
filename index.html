So-so (草々) is a poetry game that can be run on Nintendo® Game Boy™. Each game map is a poem.

It was developed using GB Studio, managed using Git, assisted with Freeform for game flow management, and generated the required bitmap text assets using Python. With the help of GB Studio’s built-in web export feature, it can also be played in a browser.

The game and the web page used the 8x8 bitmap font MisekiBitmap , ensuring that each Chinese character’s size conveniently matches one tile. With this correspondence, events could be placed on characters, triggering scene transitions when player move onto them, creating a gameplay mechanic reminiscent of “hyperlink”.
