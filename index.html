kusaï

So-so (草々) is a Chinese poetry game that can be run on Nintendo® Game Boy™. Each game map is a poem.

It was developed using GB Studio engine, managed using Git for project management, and Freeform for game flow management, and generated the required bitmap text assets using Python. Utilizing GB Studio’s built-in web export feature, it can also be played in a browser.

The game and the web page used the 8x8 bitmap font, MisekiBitmap, ensuring that the size of each Chinese character conveniently aligned with one tile. With this correspondence, events could be placed on characters, triggering scene transitions when player move onto them, creating a gameplay mechanic reminiscent of “hyperlink”.

Apple’s new whiteboard app, Freeform, was used to manage the relations (interactable keywords) between levels (poems). Each poem was generated as a game map using a Python script and then imported into the GB Studio project. The script and project files are open-source and hosted on GitHub.
